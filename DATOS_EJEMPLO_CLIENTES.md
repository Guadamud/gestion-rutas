## Instrucciones de Uso

### Para registrar los clientes:
1. Ingresar a: https://gestion-rutas-gamma.vercel.app/register
2. Llenar el formulario con los datos de cada cliente
3. El sistema los creará automáticamente como rol "cliente"

### Para agregar buses (como admin):
1. Iniciar sesión como administrador
2. Ir a "Gestión de Buses"
3. Agregar cada bus con su número y placa

### Para agregar conductores (como admin):
1. Ir a "Gestión de Conductores"
2. Registrar cada conductor con sus datos
3. Asignar cada conductor a su bus correspondiente
4. El sistema creará automáticamente su cuenta de usuario con rol "conductor"

### Para que los conductores inicien sesión:
1. Usar su email y contraseña (Conductor123!)
2. Accederán al panel de conductor donde pueden ver sus frecuencias y saldo

### Para asignar bus a cliente:
1. Ir a "Gestión de Clientes"
2. Seleccionar el cliente
3. Asignar el bus correspondiente según esta tabla
